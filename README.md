# Light Game

Have you ever played one of those puzzle games where you need to turn off all the lights, but every time you click a light it switches the status of the adjacent ones? It's a fun challenge. This app started as a way to learn the ropes of React back in 2020. 6 years later, I decided to utilize Claude to refurbish the project. All puzzles generated are solvable.

## Getting Started

To run locally clone the repo and run `npm run dev`. You must have npm installed.
